Ocultar “Añadir al carrito” is a plugin for WordPress that deletes the “Add to cart” button on the Single Product Pages in WooCoomerce.

== Description ==

Ocultar “Añadir al Carrito” es un plugin que eliminar el botón de “Añadir al Carrito” de la página de producto de WooCommerce. Es un plugin muy simple y gratuito que cumple con su función.

= Features: =
* Delete “Add to Cart” Button

== Installation ==

1. Upload the `ocultar_añadir_al_carrito.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
